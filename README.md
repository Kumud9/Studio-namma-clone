# 🎨 Studio Namma Clone

A responsive front-end clone of the **Studio Namma** website built using **HTML, CSS, and JavaScript**. This project recreates the original website's modern design, smooth animations, and interactive user experience while strengthening front-end development skills.

## 🌐 Live Demo

**Vercel:**  https://studio-namma-clone-one.vercel.app/

## 📂 GitHub Repository

https://github.com/Kumud9/Studio-namma-clone

## ✨ Features

* 🎯 Responsive design for desktop, tablet, and mobile devices
* 🎨 Modern and visually appealing user interface
* ⚡ Smooth page transitions and animations
* 📜 Scroll-based interactive effects
* 🖱️ Dynamic navigation menu
* 🎥 Embedded background video using Vimeo
* 🖼️ Animated GIF interactions
* 🌙 Dark-themed interface
* 📱 Cross-browser compatibility

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository

git clone https://github.com/Kumud9/Studio-namma-clone


2. Navigate to the project folder

cd studio-namma-clone


3. Open `index.html` in your browser.
